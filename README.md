# OpenTimestamps PHP

A complete PHP implementation of the OpenTimestamps protocol for creating and verifying blockchain timestamps.

## Features

- ✅ Create timestamp proofs for files
- ✅ Verify timestamp proofs against multiple blockchains
- ✅ Support for Bitcoin, Litecoin, and Ethereum attestations
- ✅ Advanced verification with Merkle path validation
- ✅ Configurable caching system (File, Redis, Memcached)
- ✅ Comprehensive input validation and security
- ✅ Calendar server integration
- ✅ Both detached (.ots) and attached timestamp files

## Installation

```bash
composer require opentimestamps/opentimestamps-php
